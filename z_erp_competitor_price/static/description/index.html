Overview

"This Module will help to Manage the Competitor Master along with the Pricing History of the products..!"

Using this module, Competitor Master Data we can maintain Separately along with the Customer / Vendor. For Creation of New Competitor Goto : Sales --> Competitor --> Competitor

While create from the competitor Menu automatically Is a Competitor boolean field will be enabled..!

For Customers / Vendors also if we enable Is a Competitor means it will treat as well as a competitor..!

By using pricing source we can maintain via which medium we got this price, and it will be easy to maintain the history and analysis purpose..!

For Creating pricing source : Goto : Sales --> Configuration --> Pricing Source then click create and save.

For Creating Competitor pricing : Goto : Sales --> Competitor --> Competitor Price then click create and fill the required details.

Once records are saved we have an option to group by based on Products / Competitors / Source //etc

From the competitors records itself we can see the history of the price..!

From the Product records itself we can see the history of the competitors price..!
